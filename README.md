# 项目名称

NickMagicBox
用于Java基础的学习和研究

## 项目功能
- 功能1 ：MethodInspect 运行项目后，通过反射机制，读取 src/works/ 下的　＠Argument　注解，运行注解下的方法
- 功能2 ：Printer 彩色控制台输出，分 INFO WARNNING ERROR 显示运行过程
- 功能3 ：Argument 自定义注解，用于标记待执行方法，记录方法执行时所需要的参数

## 环境要求
- Java 版本：例如 Java 11 以上
- 操作系统：例如 Windows, Linux
